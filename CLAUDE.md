# CLAUDE.md — คู่มือสำหรับ AI ช่วยพัฒนาเกมน้องงูสายรุ้ง

ไฟล์นี้ช่วยให้ Claude (หรือ AI อื่น) เข้าใจโปรเจกต์นี้อย่างรวดเร็วก่อนช่วยแก้ไข/เพิ่มฟีเจอร์

---

## โปรเจกต์นี้คืออะไร
เกมงู (Snake Game) สไตล์พาสเทล ชื่อ **น้องงูสายรุ้งพาสเทล** ไม่มี framework ไม่มี build step เปิดเล่นด้วยเบราว์เซอร์ได้เลย
- เวอร์ชันปัจจุบัน: **v1.5.8**
- ผู้พัฒนา: กัน & เจน
- เดิมอยู่ในไฟล์เดียว ~2,800 บรรทัด **ตอนนี้แยกเป็น `index.html` + `styles.css` + โฟลเดอร์ `js/` (15 ไฟล์)** เพื่อให้ดูแลง่าย

## โครงสร้าง JS (สำคัญ — อ่านก่อนแก้)
- ไฟล์ JS เชื่อมกันผ่าน **ตัวแปรส่วนกลาง (global)** ไม่ได้ใช้ระบบ module (import/export) — เพราะต้องเปิดเล่นด้วยดับเบิลคลิกไฟล์ได้
- **ลำดับ `<script>` ใน `index.html` สำคัญ:** `config.js` (ตัวแปร+canvas) ต้องมาก่อนเสมอ, `main.js` (boot) ปิดท้ายเสมอ
- จะแก้เรื่องไหน เปิดไฟล์ที่เกี่ยว: config(ตัวแปร) / audio(เสียง) / helpers / ui / score / spawn / egg / ai / effects / render(วาด) / gameover / freeroam / game(สมองหลัก: loop+update+init) / input(คีย์+สูตรลับ) / main(boot)
- **ถ้าเพิ่มตัวแปร global ใหม่** ให้ประกาศใน `config.js` / **ถ้าเพิ่มไฟล์ JS ใหม่** อย่าลืมเพิ่ม `<script src>` ใน `index.html` ตามลำดับ dependency

## ก่อนแก้โค้ด อ่านนี่ก่อน
- 📖 **`GAME_LOGIC.md`** — อธิบาย logic เกมทั้งหมดแบบละเอียด + แผนผังว่าระบบไหนอยู่ไฟล์ไหน **อ่านก่อนเสมอ** ก่อนแตะโค้ด
- เกมซับซ้อนกว่าที่เห็น มีหลายระบบเชื่อมโยงกัน (อึ/พิษ/ไข่/หิว/AI/คอมโบ/Grace Period)

## กฎการทำงานกับเจ้าของโปรเจกต์ (พี่กัน)

> เจ้าของชื่อ **กัน** เป็นเภสัชกร **ไม่มีพื้นฐานการเขียนโค้ด** แต่เข้าใจ logic ได้ดีมาก

- **คุยเป็นภาษาไทยแบบบ้านๆ** ไม่เป็นทางการ ไม่ใช้ศัพท์เทคนิคโดยไม่อธิบาย
- **อธิบาย logic สำคัญทุกครั้ง** ที่เขียน/แก้โค้ด เผื่อให้เข้าใจว่ามันทำงานยังไง
- **เวลาแก้บั๊ก ส่งไฟล์เต็มเสมอ** ไม่ใช่แค่บางส่วน
- **ทำทีละขั้น** รอเจ้าของบอก "โอเค" ก่อนไปต่อ
- **ห้ามลบฟีเจอร์** โดยไม่ถามก่อน
- **ห้าม push ขึ้น git โดยไม่ถาม** ต้องถามยืนยันทุกครั้ง

### ภาษาใน UI (สำคัญ)
- ข้อความใน UI (ปุ่ม/หัวข้อ/popup) **ห้ามใช้เครื่องหมาย `?`** เช่น ใช้ "ลืมรหัสผ่าน" ไม่ใช่ "ลืมรหัสผ่าน?"
- ใช้ภาษาไทยเป็นหลัก คำไหนแปลไทยได้ให้แปล

## Stack & การ Deploy
- **เทคโนโลยี:** HTML5 Canvas, Vanilla JavaScript, Web Audio API, localStorage
- **เก็บโค้ด:** GitHub
- **Deploy:** Cloudflare Pages — ไฟล์ `index.html` บนสุดถูกเปิดเป็นหน้าแรกอัตโนมัติ

## เวลาเปลี่ยนเวอร์ชัน
- เลขเวอร์ชันอยู่ใน `index.html` 2 จุด:
  1. `<title>` ในส่วน `<head>` (เช่น `<title>น้องงูสายรุ้ง Version 1.5.8</title>`)
  2. `<div class="version-tag">` ท้ายหน้า (เช่น `Version: 1.5.8 | Developed by กัน & เจน`)
- **ห้ามขึ้นเวอร์ชันเองอัตโนมัติ** รอเจ้าของสั่ง
