# 🧠 อธิบาย Logic เกมน้องงูสายรุ้งพาสเทล (ฉบับละเอียด)

เอกสารนี้อธิบายการทำงานภายในของเกมทั้งหมด สำหรับใช้อ้างอิงตอนแก้ไข/เพิ่มฟีเจอร์ในอนาคต
เกมแบ่งเป็น 3 ส่วน: โครงหน้าเว็บ (`index.html`), หน้าตา (`styles.css`), สมองเกม (โฟลเดอร์ `js/` 15 ไฟล์)

> **แต่ละระบบในเอกสารนี้อยู่ไฟล์ไหน:** ภาพรวม `config.js` / เสียง `audio.js` / งูผู้เล่น+update+init `game.js` / อาหาร-คะแนน-คอมโบ `game.js`+`spawn.js` / อึ-พิษ `game.js` / หัวใจ `game.js`+`spawn.js` / ไข่-ลูกงู `egg.js`+`spawn.js` / หิว `game.js` / หิน(ด่าน2) `helpers.js`+`spawn.js` / AI(ด่าน4) `ai.js` / Free Roam `freeroam.js` / สูตรลับ `input.js` / วาดภาพ `render.js`+`effects.js`+`gameover.js` / กระดานคะแนน `score.js` / ปุ่มควบคุม `ui.js`+`input.js`

> **อ่านง่ายๆ:** ลองนึกว่ากระดานเกมคือตาราง 20×20 ช่อง แต่ละช่องกว้าง 20 พิกเซล (รวมเป็นจอ 400×400)
> ทุกอย่างในเกม (งู อาหาร หิน อึ) อยู่ในระบบพิกัดช่อง (x, y) ตั้งแต่ 0 ถึง 19

## 1. ภาพรวมการทำงาน (หัวใจของเกม)

### วงจรเกม (Game Loop)
เกมทำงานเป็น "รอบ" ซ้ำๆ ไม่หยุด เรียกว่า `mainLoopTrigger()` แต่ละรอบห่างกันตามความเร็วที่เลือก (`currentSpeed` หน่วยมิลลิวินาที):

```
รอบหนึ่ง = update() → updateAIs() → updateEgg() → draw() → นัดรอบถัดไป
```

- `update()` = ขยับงูผู้เล่น 1 ก้าว + เช็คทุกการชน/การกิน
- `updateAIs()` = ขยับงู AI (เฉพาะด่าน 4)
- `updateEgg()` = ขยับไข่/ลูกงู
- `draw()` = วาดทุกอย่างลงจอใหม่

### สถานะเกม (gameState)
เกมมี 5 สถานะ ที่คุมว่าตอนนี้ทำอะไรได้บ้าง:
| สถานะ | ความหมาย |
|---|---|
| `PLAYING` | กำลังเล่นปกติ |
| `PAUSED` | พักเกม (กด Spacebar) |
| `DEAD` | ตายแล้ว รอเริ่มใหม่ |
| `FREEROAM` | งูยาวเต็มจอ พังกรอบออกมาวิ่งนอกจอ |
| `AI_VICTORY_LAP` | (ด่าน 4) ผู้เล่นโดน AI ฆ่า กำลังรอ AI วิ่งฉลอง |

### ความเร็ว (Speed)
- มี 5 ระดับ: คลาน(200) → เดิน(150) → กำลังดี(110) → ซิ่ง(70) → เดอะแฟลช(40) — ตัวเลขคือมิลลิวินาทีต่อก้าว (น้อย = เร็ว)
- `baseSpeed` = ความเร็วที่เลือกไว้, `currentSpeed` = ความเร็วจริงตอนนี้ (ด่าน 3 จะเร่งขึ้นเรื่อยๆ ไม่เท่า baseSpeed)

## 2. งูผู้เล่น (Snake)

### โครงสร้าง
งูคือ **array ของช่อง** `snake[]` โดย `snake[0]` = หัว, ตัวสุดท้าย = หาง
แต่ละข้อมี: `{ x, y, digesting, bruiseTimer, bloodTimer }`
- `digesting` = ข้อนี้กำลัง "ป่อง" เพราะเพิ่งกินอาหาร (วาดให้ใหญ่ขึ้น)
- `bruiseTimer` = เวลานับถอยหลังรอยช้ำ (ตอนกัดตัวเอง)
- `bloodTimer` = เวลานับถอยหลังรอยเลือด (ตอนชนหิน)

### การขยับ (หลักการพื้นฐานของเกมงู)
ทุกก้าว: เพิ่มหัวใหม่ข้างหน้า (`unshift`) + ตัดหางออก (`pop`)
- ถ้า**กินอาหาร** → ไม่ตัดหาง (งูยาวขึ้น 1)
- ถ้า**ไม่กิน** → ตัดหาง (ยาวเท่าเดิม)

### ทิศทาง
- `dx, dy` = ทิศที่กำลังจะไป (เช่น ขวา = dx:1, dy:0)
- `lastDx, lastDy` = ทิศล่าสุดที่เดินจริง (ใช้ตอนหยุดนิ่งเพื่อรู้ว่าหัวหันทางไหน)
- **ห้ามถอยหลังกลับ** — ถ้ากำลังไปขวา จะกดซ้ายทันทีไม่ได้ (จะกินหัวตัวเอง)

### Grace Period (ระบบผ่อนผันตอนจะชน) — สำคัญและซับซ้อน
เมื่อหัวงูกำลังจะชน (กำแพง/หิน/ตัวเอง) เกม**ไม่หักพลังทันที** แต่ให้ "เบรกค้าง" ไว้ก่อน:
1. งูหยุดนิ่ง (dx=0, dy=0) ที่ขอบของสิ่งที่จะชน — ยังไม่เสียพลัง
2. ผู้เล่นมีเวลา **0.5 วินาที** หักเลี้ยวหนีได้ → ถ้าหนีทัน ไม่เสียอะไรเลย
3. ถ้าไม่หนี (หรือฝืนกดทิศเดิม) → ครบ 0.5 วิ ค่อยหักพลัง -1 และเล่นเอฟเฟกต์สั่น/เปลี่ยนสี

ตัวแปรที่เกี่ยว: `inGracePeriod`, `pendingCollisionDamage`, `collisionDir`, `graceTimeout`

## 3. ระบบอาหาร คะแนน และคอมโบ

### อาหาร (food)
- มี 1 ชิ้นบนจอเสมอ เป็นวงกลมสีชมพู
- กินแล้วได้ **5 คะแนน** (+ โบนัสคอมโบ) งูยาวขึ้น 1 และเกิดอาหารใหม่ทันที
- `foodEatenCount` = นับอาหารที่กินสะสม (ใช้คุมหลายระบบ: สีตา, วางไข่, อึ, ความเร็วด่าน 3)

### คอมโบ (Combo)
- ถ้ากินอาหารชิ้นถัดไปภายใน **3 วินาที** → คอมโบ +1
- โบนัสคะแนน = `5 × (คอมโบ × 0.5)` ยิ่งคอมโบสูงยิ่งได้เยอะ
- ตอนคอมโบ งูจะเรืองแสงสีรุ้ง + มีตัวหนังสือ "COMBO x?" เด้งขึ้น
- พักเกมไม่ทำให้คอมโบหาย (โค้ดชดเชยเวลาที่หยุดไว้)

### สีตางู (โตขึ้นตามประสบการณ์)
ทุกๆ 15 อาหาร ตางูเปลี่ยนสี ไล่จาก เทา → ฟ้า → น้ำตาล → ส้ม → แดง → แดงเข้ม (6 ระดับ)

## 4. ระบบอึและพิษ (Poop & Toxic) — เอกลักษณ์ของเกมนี้

### การขับถ่าย
- เริ่มทำงานหลังกินอาหารครบ 5 คำ
- **ทุกๆ 5 คำ** (คำที่ 5, 10, 15...) งูขับอึออกมาที่หาง จำนวน `(2 + ความยาว/5) × ตัวคูณท้องเสีย`
- คำอื่นๆ ที่ไม่ใช่ทุก 5 ก็มีโอกาสสุ่มขับอึตามความยาวงู
- อึมีอายุ 60 ก้าว แล้วจางหายเอง (สีค่อยๆ เปลี่ยน)

### กินอึ (อันตราย)
- เผลอกินอึ → งู**สั้นลง 2 ข้อ** + `poopsEaten` +1 (ตัวนับพิษ)
- พิษแสดงเป็นสถานะ: สะอาด → เริ่มปวดท้อง → พะอืดพะอม → ท้องเสีย → วิกฤต
- **พิษครบ 5 → งูระเบิดตาย!**
- ยิ่งพิษเยอะ ยิ่งท้องเสีย (ตัวคูณ `diarrheaMult` ทำให้ขับอึถี่ขึ้น = วงจรอุบาทว์)
- กินหัวใจ ❤️ จะ**ล้างพิษ**กลับเป็น 0

### กฎพิเศษ (มุกตลก)
ถ้ากำลังเรืองแสงหัวใจ (เพิ่งกินหัวใจ) แล้วไปกินอึ → ระเบิดตายทันที พร้อมข้อความ **"ใครเขากินอึกัน"** 😂

## 5. ระบบหัวใจ (Heart) ❤️

- มีโอกาสสุ่มเกิดหัวใจหลังกินอาหาร (`dropRate` ~30% ปรับตามความเร็ว — เร็วมาก โอกาสน้อยลง; พลังเต็มแล้ว โอกาสลดครึ่ง)
- หัวใจมีอายุ 50 ก้าว (ใกล้หมดอายุจะสั่น) แล้วหายไป
- กินหัวใจ: ถ้าพลังยังไม่เต็ม (< 10) → พลัง +1 / ถ้าเต็มแล้ว → +11 คะแนนแทน
- กินหัวใจ**ล้างพิษเป็น 0** เสมอ
- พลังสูงสุด `MAX_LIVES = 10` เริ่มเกมมี 3

## 6. ระบบไข่และลูกงู (Egg & Baby) 🥚 — น่ารักที่สุด

### วางไข่
- ทุกๆ 100 อาหาร (และไม่ใช่ด่าน 4) งูวางไข่ 1 ฟอง
- **ห้ามกินไข่!** ถ้ากินไข่ → เสียพลัง -4 และงูสั้นลงครึ่งตัว (เศร้า)

### ไข่ฟัก
- ไข่สั่นๆ รอ 150 ก้าว แล้วฟักเป็น **ลูกงู (BABY)**

### ลูกงู (AI เล็กๆ)
- ลูกงูเดินหาอาหารกินเองด้วย AI ง่ายๆ (เลือกทางที่ปลอดภัยและใกล้อาหารสุด)
- บางครั้งวิ่งเข้าหาพ่อแม่ (ผู้เล่น) ด้วยความรัก ❤️ (สุ่ม 20%)
- **ห้ามกินลูกงู!** ถ้ากิน → เสียพลัง -7 และงูสั้นเหลือ 20% (เศร้ามาก)
- ลูกงูกินอาหารครบ 10 คำ → "โตแล้ว" วิ่งหนีออกขอบจอไปใช้ชีวิต (escaping)

## 7. ระบบหิว (Starvation) 💀

ป้องกันไม่ให้ผู้เล่นวนหนีไม่กินอาหาร:
- ไม่กินอาหารเกิน **100 ก้าว** → เข้าโหมดหิว (`isStarved`)
- งูเริ่ม**ผอมโทรม**: ตัวจางลง สั่น และ**สั้นลงเรื่อยๆ** + พลังค่อยๆ ลด
- ยิ่งหิวนาน งูยิ่งจางจนเกือบมองไม่เห็น
- ถ้าหดจนเหลือหัวเดียว → กลายเป็น**หัวกะโหลก** (`isSkullPhase`) นับถอยหลัง 15 ก้าว แล้ว **"อดตาย"** 💀
- กินอาหารเมื่อไหร่ รีเซ็ตตัวนับหิวกลับเป็น 0 ทันที

## 8. ด่านที่ 2: สิ่งกีดขวาง (Obstacles)

### การสร้างหิน
- สุ่มสร้างก้อนหิน 6–10 ก้อน แต่ละก้อนเป็นรูปทรงสุ่มในกรอบ 3×3 ถึง 5×5
- มีการตรวจสอบฉลาดมาก เพื่อไม่ให้เกมพัง:
  - **`isLocalConnected`** — เซลล์ในก้อนต้องติดกันเป็นรูปเดียว
  - **`isValidEmptySpace`** — ห้ามมีช่องว่างถูกขังด้านใน และห้ามมีทางแคบ 1 ช่อง (กันงูติดตาย)
  - เว้น "safe zone" รอบจุดเกิดงู และไม่ทับอาหาร
  - หินแต่ละก้อนต้องห่างกันอย่างน้อย 1 ช่อง
- กดปุ่ม "สุ่มลายหิน" หรือ `Shift` สุ่มใหม่ได้ (เฉพาะก่อนออกเดิน)

### ทุบหินแตก
- ชนหินเสียพลัง -1 และหินมี "รอยร้าว" (`hitCount`) เพิ่มขึ้น สีเข้มขึ้นเรื่อยๆ
- ชนหินช่องเดิมครบ **5 ครั้ง** → ช่องนั้น**แตกหาย** เหลือรอยเลือดถาวร
- ถ้าทุบจนก้อนหินขาดเป็น 2 ส่วน เกมจะแยกเป็น 2 ก้อน (`getConnectedGroups`)

## 9. ด่านที่ 4: ศึกชิงแชมป์ (AI Battle) 🤖

### ระบบคลื่นศัตรู (Wave)
- AI เกิดจากขอบจอเมื่อผู้เล่นเริ่มเดิน เริ่มจาก 1 ตัว → เพิ่มทีละตัวจนถึง 5 → วนกลับมา 1
- ตัวเรายาวได้สูงสุด **20 ช่อง** (`MAX_SNAKE_LENGTH_AI`)

### AI 6 พฤติกรรม (เปลี่ยนทุก 30 ก้าว)
| พฤติกรรม | ทำอะไร |
|---|---|
| `HUNT` | ไล่ล่าหัวผู้เล่น |
| `FOOD` | วิ่งไปกินอาหาร |
| `STEAL` | แย่งอาหาร (ตัวเล็ก) |
| `FLEE` | หนีผู้เล่น (ตัวสั้นกว่า) |
| `DODGE` | หลบผู้เล่น (ตัวยาวกว่า) |
| `ROAM` | เดินเล่นสุ่ม (มีโอกาสเกิด 1 ตัวต่อคลื่นถ้า ≥3 ตัว) |

ตรรกะการเลือก: ตัวเล็ก(≤3)→หนีหรือแย่งอาหาร / ยาวกว่าเรา→40%ล่า 40%อาหาร 20%หลบ / สั้นกว่าเรา→หนี
- ความเร็ว AI ผันตามความยาว (ตัวสั้นเร็ว 1.5x, ตัวยาวช้าลงเหลือ 1.0x) — `getAiSpeed`

### การต่อสู้ (ใครชนหัวใคร)
- **เราเอาหัวพุ่งชนตัว AI** → AI สั้นลง 1
- **AI เอาหัวชนตัวเรา** → เราสั้นลง 1
- **หัวชนหัว** → ทั้งคู่สั้นลง 1
- **ทำ AI สั้นจนตาย** → +20 คะแนน, +2 พลัง, ยาวขึ้น 3 ช่อง! 🏆
- **AI ชนเราจนเหลือหัวเดียว** → ผู้เล่นตาย AI วิ่งฉลอง (Victory Lap) ขึ้น "HAHA!" แล้วจบเกม "โดน AI ฆ่าตาย"

> ด่านนี้**ไม่มีระบบวางไข่** — บู๊อย่างเดียว

## 10. โหมดพังกรอบ (Free Roam) 🌈

- เมื่องูยาวจน**เต็มจอ** (เกือบทุกช่อง) → ชนะด่าน! งู "พังกรอบ" ออกมาวิ่งเล่น**นอกจอเกม**ทั่วหน้าเว็บ
- ตอนนี้งูกลายเป็น HTML element ลอยทับทั้งหน้าจอ (`freeRoamContainer`) บังคับทิศต่อได้
- วิ่งทะลุขอบหน้าจอไปโผล่อีกฝั่งได้
- เอาหัวชนปุ่ม "รีเซ็ตเกม" เพื่อเริ่มใหม่

## 12. ระบบเสริมอื่นๆ

### เสียงและดนตรี (Web Audio API)
- เสียงเอฟเฟกต์ (กิน/ชน/หัวใจ) สังเคราะห์สดด้วย `playSound()`
- เพลง Lo-fi เป็นคอร์ด 4 ชุดวนซ้ำทุก 4 วินาที (`startLofiMusic`) เปิด/ปิดได้

### กระดานคะแนน (Leaderboard)
- เก็บ 5 อันดับลง `localStorage` (key: `pastelSnakeScores`)
- บันทึก: ชื่อ, คะแนน, ก้าวเดิน, จำนวนอึที่กิน, จำนวนหัวใจที่กิน

### เอฟเฟกต์ภาพ
- `sparkles` = ประกายตอนกิน, `floatingTexts` = ตัวหนังสือเด้ง, `explosionParticles` = เศษงูตอนระเบิด
- งูมีสีหน้าหลายแบบ: ยิ้ม(กินอาหาร), ตาเอ็กซ์(กินอึ), เศร้า(กินไข่/ลูก), หัวใจ(มีลูกงู), หัวกะโหลก(อดตาย)

### รองรับมือถือ
- ปุ่มทิศทาง (D-Pad) บนจอ + ปุ่มสลับ layout คอม/มือถือ
- กันหน้าจอเลื่อนตอนแตะปุ่ม (`touchmove` preventDefault)
