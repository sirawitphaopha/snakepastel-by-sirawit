# 🐍 น้องงูสายรุ้งพาสเทล (Pastel Rainbow Snake)

เกมงู (Snake Game) สไตล์พาสเทลน่ารัก เล่นบนเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร — เป็นไฟล์ HTML ไฟล์เดียวจบ (HTML + CSS + JavaScript ล้วน ไม่ใช้ framework)

> **เวอร์ชันปัจจุบัน:** v1.5.8
> **ผู้พัฒนา:** กัน & เจน

---

## ✨ จุดเด่น

- 🎮 **5 ด่าน** แต่ละด่าน logic ต่างกันสิ้นเชิง
- 💩 **ระบบอึ/พิษ** — งูกินอาหารมากไปก็ขับถ่าย เผลอกินอึตัวเองสั้นลง พิษครบ 5 ตาย
- 🥚 **ระบบไข่และลูกงู** — กินครบ 100 คำงูวางไข่ ฟักออกมาเป็นลูกงูที่เดินหาอาหารเอง
- ❤️ **หัวใจฟื้นพลัง + ระบบคอมโบ** — กินอาหารรัวๆ ได้โบนัสคะแนน
- 💀 **ระบบหิว** — ไม่กินอาหารนานๆ งูจะผอมโทรมจนกลายเป็นหัวกะโหลกแล้วตาย
- 🤖 **AI คู่แข่ง** ในด่าน 4 — งู AI มี 6 พฤติกรรม (ล่า/หนี/หลบ/แย่งอาหาร/เดินเล่น)
- 🏆 **กระดานคะแนน (Leaderboard)** เก็บลงเครื่อง 5 อันดับ
- 🎵 **เพลง Lo-fi** สังเคราะห์สดในเบราว์เซอร์
- 📱 **เล่นได้ทั้งคอมและมือถือ** มีปุ่มทิศทาง (D-Pad) บนจอ
- 🤫 **สูตรลับ 6 รหัส** ซ่อนไว้ในเกม

---

## 🚀 วิธีเล่น

### เล่นในเครื่อง
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ (Chrome, Edge, Firefox, Safari) ได้เลย

### ปุ่มควบคุม
| ปุ่ม | การทำงาน |
|---|---|
| ลูกศร ⬆️⬇️⬅️➡️ หรือ `W` `A` `S` `D` | บังคับทิศทาง |
| `Spacebar` | พักเกม / เล่นต่อ |
| `Enter` | เริ่มเกมใหม่ (ตอนตาย) |
| `Shift` | สุ่มลายหินใหม่ (เฉพาะด่าน 2 ก่อนออกเดิน) |
| ปุ่มบนจอ (D-Pad) | บังคับทิศบนมือถือ |

## 🎯 5 ด่าน

| ด่าน | ชื่อ | กติกา |
|---|---|---|
| 0 | ดั้งเดิม | มีกำแพง ชนขอบจอหรือตัวเองตาย |
| 1 | วาร์ปทะลุกำแพง | เลื้อยทะลุขอบจอไปโผล่อีกฝั่งได้ |
| 2 | สิ่งกีดขวางสุ่ม | มีก้อนหินสุ่มกระจายทั่วจอ ชนตาย (ทุบหินแตกได้) |
| 3 | ยิ่งกินยิ่งซิ่ง | ความเร็วเพิ่มขึ้นทุก 30 อาหาร (สูงสุด 15 ระดับ) |
| 4 | ศึกชิงแชมป์ | เจองู AI เป็นคลื่น 1→5 ตัว ตัวเรายาวสุด 20 ช่อง |

## 🛠️ เทคโนโลยี

- **HTML5 Canvas** — วาดเกมทั้งหมด
- **Vanilla JavaScript** — logic ล้วน ไม่มี library
- **Web Audio API** — เสียงเอฟเฟกต์และเพลง Lo-fi สังเคราะห์สด
- **localStorage** — เก็บกระดานคะแนน
- ทั้งหมดอยู่ใน **ไฟล์เดียว** (`index.html`)

## 📂 ไฟล์ในโปรเจกต์

> เดิมเกมอยู่ในไฟล์ HTML ไฟล์เดียว ~2,800 บรรทัด ตอนนี้แยกออกเป็นหลายไฟล์เพื่อให้ดูแลง่ายขึ้น (ยังเปิดเล่นด้วยเบราว์เซอร์ได้เหมือนเดิม)

| ไฟล์ | คือ |
|---|---|
| `index.html` | โครงหน้าเว็บ + ลิงก์ไฟล์ CSS/JS (~100 บรรทัด) |
| `styles.css` | หน้าตา/สไตล์ทั้งหมด |
| `js/` | โค้ดเกมทั้งหมด แยกเป็น 15 ไฟล์ตามระบบ (ดูตารางด้านล่าง) |
| `README.md` | ไฟล์นี้ — ภาพรวมโปรเจกต์ |
| `GAME_LOGIC.md` | อธิบาย logic เกมทั้งหมดแบบละเอียด |
| `CLAUDE.md` | คู่มือให้ AI (Claude) เข้าใจโปรเจกต์ตอนช่วยพัฒนา |

### โฟลเดอร์ `js/` (โหลดตามลำดับนี้ใน `index.html`)

| ไฟล์ | ดูแลเรื่อง |
|---|---|
| `config.js` | ค่าคงที่ + ตัวแปรเกมทั้งหมด + canvas |
| `audio.js` | เสียงเอฟเฟกต์ + เพลง Lo-fi |
| `helpers.js` | ฟังก์ชันช่วย + ตรวจการชน + สร้างรูปหิน |
| `ui.js` | สลับจอ/พักเกม/ความเร็ว/บังคับทิศทาง |
| `score.js` | กระดานคะแนน + สถานะ + วิธีเล่น |
| `spawn.js` | เกิดอาหาร/หัวใจ/หิน/ไข่ |
| `egg.js` | ระบบไข่ + ลูกงู |
| `ai.js` | ระบบ AI (ด่าน 4) |
| `effects.js` | ประกาย + ตัวหนังสือเด้ง |
| `render.js` | วาดฉาก + วาดงู |
| `gameover.js` | จอจบเกม + ระเบิดงู |
| `freeroam.js` | โหมดพังกรอบวิ่งนอกจอ |
| `game.js` | สมองหลัก: วงจรเกม + เดิน/กิน/ชน + เริ่มเกม |
| `input.js` | คีย์บอร์ด + D-Pad + สูตรลับ |
| `main.js` | จุดเริ่มเกมตอนเปิดหน้า (boot) |

> **หมายเหตุ:** ไฟล์ JS เชื่อมกันผ่านตัวแปรส่วนกลาง (global) ลำดับ `<script>` ใน `index.html` จึงสำคัญ — `config.js` ต้องมาก่อนเสมอ และ `main.js` ปิดท้ายเสมอ

## 🌐 การ Deploy

โปรเจกต์นี้ deploy ผ่าน **Cloudflare Pages** — Cloudflare อ่านทั้งโฟลเดอร์ และเปิด `index.html` ที่อยู่บนสุดเป็นหน้าแรกอัตโนมัติ ไฟล์ CSS/JS ที่แยกออกมาถูกโหลดตามลิงก์ใน `index.html` เอง ไม่ต้องตั้งค่าหรือ build อะไรเพิ่ม
