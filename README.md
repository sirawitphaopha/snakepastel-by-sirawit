# 🐍 น้องงูสายรุ้งพาสเทล (Pastel Rainbow Snake)

เกมงู (Snake Game) สไตล์พาสเทลน่ารัก เล่นบนเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร — เป็นไฟล์ HTML ไฟล์เดียวจบ (HTML + CSS + JavaScript ล้วน ไม่ใช้ framework)

> **เวอร์ชันปัจจุบัน:** v1.5.8
> **ผู้พัฒนา:** กัน & เจน

---

## ✨ จุดเด่น

- 🎮 **5 ด่าน** แต่ละด่าน logic ต่างกันสิ้นเชิง
- 💩 **ระบบอึ/พิษ** — งูกินอาหารมากไปก็ขับถ่าย เผลอกินอึตัวเองสั้นลง พิษครบ 5 ตาย
- 🥚 **ระบบไข่และลูกงู** — กินครบ 100 คำงูวางไข่ ฟักออกมาเป็นลูกงูที่เดินหาอาหารเอง
- ❤️ **หัวใจฟื้นพลัง + ระบบคอมโบ** — กินอาหารรัวๆ ได้โบนัสคะแนน
- 💀 **ระบบหิว** — ไม่กินอาหารนานๆ งูจะผอมโทรมจนกลายเป็นหัวกะโหลกแล้วตาย
- 🤖 **AI คู่แข่ง** ในด่าน 4 — งู AI มี 6 พฤติกรรม (ล่า/หนี/หลบ/แย่งอาหาร/เดินเล่น)
- 🏆 **กระดานคะแนน (Leaderboard)** เก็บลงเครื่อง 5 อันดับ
- 🎵 **เพลง Lo-fi** สังเคราะห์สดในเบราว์เซอร์
- 📱 **เล่นได้ทั้งคอมและมือถือ** มีปุ่มทิศทาง (D-Pad) บนจอ
- 🤫 **สูตรลับ 6 รหัส** ซ่อนไว้ในเกม

---

## 🚀 วิธีเล่น

### เล่นในเครื่อง
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ (Chrome, Edge, Firefox, Safari) ได้เลย

### ปุ่มควบคุม
| ปุ่ม | การทำงาน |
|---|---|
| ลูกศร ⬆️⬇️⬅️➡️ หรือ `W` `A` `S` `D` | บังคับทิศทาง |
| `Spacebar` | พักเกม / เล่นต่อ |
| `Enter` | เริ่มเกมใหม่ (ตอนตาย) |
| `Shift` | สุ่มลายหินใหม่ (เฉพาะด่าน 2 ก่อนออกเดิน) |
| ปุ่มบนจอ (D-Pad) | บังคับทิศบนมือถือ |

---

## 🎯 5 ด่าน

| ด่าน | ชื่อ | กติกา |
|---|---|---|
| 0 | ดั้งเดิม | มีกำแพง ชนขอบจอหรือตัวเองตาย |
| 1 | วาร์ปทะลุกำแพง | เลื้อยทะลุขอบจอไปโผล่อีกฝั่งได้ |
| 2 | สิ่งกีดขวางสุ่ม | มีก้อนหินสุ่มกระจายทั่วจอ ชนตาย (ทุบหินแตกได้) |
| 3 | ยิ่งกินยิ่งซิ่ง | ความเร็วเพิ่มขึ้นทุก 30 อาหาร (สูงสุด 15 ระดับ) |
| 4 | ศึกชิงแชมป์ | เจองู AI เป็นคลื่น 1→5 ตัว ตัวเรายาวสุด 20 ช่อง |

## 🛠️ เทคโนโลยี

- **HTML5 Canvas** — วาดเกมทั้งหมด
- **Vanilla JavaScript** — logic ล้วน ไม่มี library
- **Web Audio API** — เสียงเอฟเฟกต์และเพลง Lo-fi สังเคราะห์สด
- **localStorage** — เก็บกระดานคะแนน
- ทั้งหมดอยู่ใน **ไฟล์เดียว** (`index.html`)
